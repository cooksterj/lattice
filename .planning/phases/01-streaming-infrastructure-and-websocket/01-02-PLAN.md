  <action>
Add a new function `create_asset_websocket_router(manager: ExecutionManager) -> APIRouter` at the bottom of execution.py, following the same pattern as the existing `create_websocket_router()`.

The router has a single endpoint:

```python
@router.websocket("/ws/asset/{key:path}")
async def asset_websocket(websocket: WebSocket, key: str) -> None:
```

Use `{key:path}` to support grouped asset keys like `analytics/stats`.

Endpoint behavior:
1. `await websocket.accept()`
2. `manager.add_asset_subscriber(key, websocket)`
3. Send replay buffer catch-up:
   ```python
   replay = manager.get_replay_buffer(key)
   if replay:
       await websocket.send_json({
           "type": "replay",
           "data": {"entries": replay},
       })
   ```
4. Send current execution state if running:
   ```python
   if manager.is_running and manager.executor is not None:
       state = manager.executor.current_state
       if state is not None and key in {str(k) for k in state.asset_results}:
           asset_result = next(
               (r for k, r in state.asset_results.items() if str(k) == key), None
           )
           if asset_result is not None:
               await websocket.send_json({
                   "type": "asset_state",
                   "data": {
                       "asset_key": key,
                       "status": asset_result.status.value,
                       "started_at": asset_result.started_at.isoformat() if asset_result.started_at else None,
                   },
               })
   ```
5. Keep-alive loop: `while True: await websocket.receive_text()` (client does not send data; this detects disconnects).
6. `except WebSocketDisconnect: pass`
7. `finally: manager.remove_asset_subscriber(key, websocket)`

This follows the exact same structure as the existing `/ws/execution` endpoint but with per-asset scoping.

In `app.py`, import `create_asset_websocket_router` from `lattice.web.execution` (add to existing import block). In `create_app()`, after the existing WebSocket router registration, add:

```python
# Add asset-scoped WebSocket route
asset_ws_router = create_asset_websocket_router(execution_manager)
app.include_router(asset_ws_router)
```
